Lesson 06: Booleans and if-Statement in JavaScript

This exercise 6k, and 6l is to improve the cart quantity in the shopping cart. First, make a copy of the project from the previous exercise. Then, create a variable 'cartQuantity' and set it to 0. The 'cartQuantity' variable will be used to keep track of the number of items in the cart. The 'cartQuantity' variable will be updated when the user clicks on the buttons. The buttons will be used to add or remove items from the cart. The 'cartQuantity' variable will be displayed in the console when the user clicks on the buttons. The following buttons will be created:

  • Let's suppose that the cart has maximum quantity of 10. Before updating the quantity, check if the quantity will be greater than 10:
    • If it will, display a popup message saying 'The cart is full' and don't update the quantity.
    • Otherwise, update the quantity and display the console.log() it as usual.
  • In exercise 5i - 5k, we have created 'Remove from cart', '-2' and '-3' buttons, Before, updating the quantity, check if the quantity will be less than 0:
    • if it will, create a popup message saying 'Not enough items in the cart' and don't update the quantity.
    • Otherwise, update the quantity and display the console.log() it as usual.
