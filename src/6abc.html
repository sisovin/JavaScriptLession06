Lesson 06: Hour Variable in JavaScript

In this Exercise, we will create a variable called 'hour' and save the current hour of the day in it. Then we will use the 24-hour format to display the current hour of the day between 0 and 23. We use if-Statements and the condition as below:

  • if the hour is between 6 and 12, display "Good Morning!" in the console.
  • if the hour is between 13 and 17, display "Good Afternoon!" in the console.
  • Otherwise, display "Good night!" in the console.
