Lesson 06: Generate Random Number Variable in JavaScript

This exercise 6f, 6g, 6h, 6i, and 6j are about generating a random number in JavaScript. The Math.random()

  • Generate a random number with a method Math.random(). Save it in a variable.
  • Create an if-Statement and check:
    • If the random number is less than 0.5, then display 'heads' in the console.
    • Else display 'tails' in the console.
  • Let's say we're trying to guess the result. Create a variable 'guess' and save 'heads' or 'tails' in it.
    • If your guess meet the result, display 'You win!' in the console.
    • if your guess does not match the result, display 'You lose!' in the console.
  • (Challenge) Instead of using if-Statement in the previous exercise, try switching them into ternary operators (condition ? A : B) and display the result in the console.
